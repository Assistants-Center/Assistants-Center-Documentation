# Redirect URI <Badge type="warning" text="REQUIRED" />

Add to Discord-Dashboard Config:
```js
...
    redirectUri: '',
...
```

| Option      | Type   |                                                                                                      |
|-------------|--------|------------------------------------------------------------------------------------------------------|
| redirectUri | String | Discord.js Bot OAuth2 defined Redirect Uri. For discord-dashboard it's `domain.com/discord/callback` |
